# Superstore Sales Data Analysis

## Project Overview
This project performs an exploratory data analysis (EDA) of the Superstore sales dataset to generate business insights related to sales performance, profitability, customer segments, and product categories.

The analysis was conducted using Python in a Jupyter Notebook with libraries such as Pandas, Matplotlib, and Seaborn.

## Dataset
The dataset used in this project is publicly available on Kaggle:

https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

The dataset contains transactional information including order details, customer segments, geographic regions, product categories, sales values, discounts, and profit.

## Objectives
The main objectives of this analysis include:

- Understanding regional sales performance
- Identifying top-performing product categories and sub-categories
- Analyzing customer segment contributions to revenue
- Evaluating sales trends over time
- Investigating the relationship between discount and profit

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
The analysis highlights important business insights such as high-performing regions, dominant customer segments, sales growth trends, and the impact of discounts on profitability.
